================================================================================
  TROUBLESHOOTING
================================================================================

"Cannot find module" Error:
  npm install
  npm run build

Port Already in Use:
  Change port in src/main.ts (port: 3001)
  npm run build && npm start

TypeScript Error:
  npm install -D typescript @types/node
  npm run build

Server Won't Start:
  - Check Node.js: node --version (need 20+)
  - Check npm: npm --version (need 10+)
  - Try clean build: npm run clean && npm install && npm run build
