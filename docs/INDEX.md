## 💡 Key Concepts Glossary

**P0: Feedback Validation** — User provides quality feedback (1-5) that updates Q-learning  
**P1: Collaboration Detection** — System finds agent pairs that work well together  
**P2: Knowledge Extraction** — System extracts solution patterns from task outcomes  
**Q-Learning** — Reinforcement learning algorithm that optimizes agent selection  
**Audit Trail** — Immutable log of all task executions (Phase 11A)  
**SLA** — Service level agreement metrics (latency, success rate, cost)  
**Feedback Loop** — System learns from outcomes and improves routing  
**Knowledge Base** — Queryable database of solution patterns for warm-start learning  
**Agent Specialization** — Agents get better at specific task types through learning
