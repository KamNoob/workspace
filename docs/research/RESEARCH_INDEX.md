## Retrieval Protocol

**Before researching ANY topic:**

1. Search this index for existing research
2. Check status:
   - **Active & Recent** (< 30 days tech, < 90 days general): Use cached knowledge
   - **Pending Refresh**: Spawn Scout to refresh
   - **Not Found**: Spawn Scout to research fresh
3. After research delivered: Add entry to this index

**Search Tips:**
- Ctrl+F this file for keywords
- Check domain section for topic area
- Look at "Recent Research" for latest work

## How to Add Research

1. Scout researches → Veritas validates → Chronicle documents
2. Chronicle saves to: `docs/research/[domain]/[topic].md`
3. Add entry to appropriate domain section above
4. Update "Last Updated" field in that file
5. Update this index
