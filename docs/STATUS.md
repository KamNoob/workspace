## 🔧 Operations Commands

### Check System Health
```bash
# Learning system status
julia scripts/ml/unified-learning-system.jl --report

# Gateway status
openclaw status

# Recent git activity
git log --oneline -5
```

### Force Learning Cycle
```bash
# Run full P0+P1+P2 pipeline
julia scripts/ml/unified-learning-system.jl --full-cycle
```

### Record Feedback
```bash
# After task completion
julia scripts/ml/feedback-validator.jl \
  --task-id "UUID" --approved true --quality 4
```

### View Metrics
```bash
# Q-scores
jq '.task_types.code.agents | to_entries' data/rl/rl-agent-selection.json

# Feedback history
tail -5 data/feedback-logs/feedback-validation.jsonl

# Patterns
jq '.patterns | keys' data/knowledge-base/extracted-patterns.json
```
