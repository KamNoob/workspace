# 2026-02-10 - Hybrid Embedding System Setup

## Hybrid Embedding Configuration
**Implemented:** Two independent embedding systems for different purposes

### 1. memorySearch (Workspace File Search)
- **Provider:** Local GGUF (no API calls)
- **Model:** EmbeddingGemma 300M Q8 quantized
- **Location:** `~/.openclaw/models/embeddings/embeddinggemma-300M-Q8_0.gguf` (314MB)
- **Source:** Hugging Face (ggml-org/embeddinggemma-300M-GGUF)
- **Vector dimensions:** 768
- **Status:** ✅ Operational

### 2. memory-lancedb (Persistence/Real-time Memory)
- **Provider:** OpenAI (cloud)
- **Model:** text-embedding-3-small
- **Purpose:** Auto-capture/recall conversation memories
- **Status:** ✅ Operational
