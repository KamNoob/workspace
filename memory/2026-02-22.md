## Key Technical Details

### Q-Learning Formula Applied
```
Q(memory) ← Q(memory) + α[reward + γ max Q' - Q]
Where:
  α = 0.01 (learning rate)
  γ = 0.99 (discount factor)
  reward = success + context_match - lookup_time
```

### Eligibility Traces
```
e(memory) ← λγe(memory) + 1
Q ← Q + αδ·e
Credit recent facts for distant successes
```
