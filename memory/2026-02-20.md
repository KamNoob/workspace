## 🎓 Key Learnings

1. **Notion API Quirks**: Database items appear as "New page" in search (normal behavior)
2. **People Field Limitation**: Can't assign database items to agents via API; created separate "Agent" field instead
3. **Untitled Pages**: Database records don't have page titles—they use properties instead
4. **ID Persistence**: All 4 database IDs must be stored in .env for scripts to work
5. **Environment Variables**: All sync scripts need `source ~/.openclaw/.env` at start
